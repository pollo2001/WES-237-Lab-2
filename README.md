# WES-237-Lab-2

### Includes Code:
- main.c: C functions (myAdd, myProduct) with simulated hardware latency.

- libMyLib.so: Compiled shared library for Python interfacing via ctypes.

- ctypes_example.ipynb: Python wrappers for C functions.

- multiprocess_example.ipynb: Hardware parallelism using taskset to pin tasks to Core 0 and Core 1.

- Demonstrates Shared Memory using multiprocessing.Array for inter-process communication.

- threading_example.ipynb: Concurrency management.

- Deadlock Solution: Enforced lock ordering to prevent circular waiting.

- Non-blocking Acquire: Logic to maintain execution flow during resource contention.
